
# Employee Management System

## Tech Stack
- React.js
- Spring Boot
- MySQL
- Spring Security
- JWT Authentication

## Features
- Employee CRUD
- Department CRUD
- Designation CRUD
- Attendance Management
- Leave Management
- Salary Management
- JWT Authentication
- Role Based Access
- Dashboard Analytics

## Setup

### Backend
mvn spring-boot:run

### Frontend
npm install
npm run dev
